That which is to shrink must first be allowed to expand. That which is to weaken must first be made strong. That which is to be thrown down must first be raised up. That which is to be taken must first be given. This is called subtle illumination. The soft and weak overcome the hard and strong. Fish should not be taken from deep water. The state's sharp weapons should not be displayed to the people.
